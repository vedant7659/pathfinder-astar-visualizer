# PathFinder – A* Pathfinding Visualizer

Interactive visualization of the A* pathfinding algorithm implemented in C++ using SFML.

## Features
• Grid-based pathfinding environment  
• A* shortest path algorithm implementation  
• Visual representation of computed paths  
• Modular C++ architecture  

## Tech Stack
• C++
• SFML
• A* Pathfinding Algorithm

## How It Works
The program constructs a grid environment where the A* algorithm evaluates possible paths from a start node to a goal node using heuristic cost estimation.
